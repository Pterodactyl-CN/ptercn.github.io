翼龙面板

完全免费、开源的游戏服务器管理面板，支持 Minecraft、CSGO 等游戏的服务端

完全开放的 API

利用 API，您可以开发任何对接 Pterodactyl 的插件，连接您的网站，自动化出售等。

在线文件编辑工具

您无需为了修改服务器配置文件而启动SFTP软件，只需使用在线编辑器修改后保存即可。

功能齐全的控制台

基于 WebSocket 的网页命令控制台，完全自由掌控服务器，就像直接操作命令行一样。

方便的文件管理器

在您的设备上没有好用的 SFTP 软件？别担心，我们的面板集成了多功能文件管理器。

多节点服务器支持

翼龙面板支持连接到多个后端程序，您完全可以用于部署分布式服务器负载均衡节点。

可自定义启动参数

如果您的服务端需要特殊的启动参数，您完全可以在设置中指定核心JAR名称和参数。

响应式前端网页设计

我们的管理面板对于移动设备兼容性非常好，无论您是使用高清电视、个人电脑、平板还是手机访问管理面板，都可以获得极佳的使用体验。同时面板使用了 Laravel Blade 模板引擎，您可以方便快捷地制作出美观大气的面板主题。

安全性

安全永远是我们最主要的考虑，翼龙面板不使用MD5，我们使用最新的安全标准例如 BCrypt 和 AES-256-CBC

自定义

翼龙面板使用 Blade 模板引擎，您可以自由修改和创作新的 CSS，面板前端几乎任何地方都可以修改和自定义。

Docker

翼龙面板支持利用 Docker 控制服务器资源分配并加强网络控制管理的游戏面板，利用容器虚拟化保证安全性。

用户友好

翼龙面板的界面布局简洁、大方，即使您以前没有使用过面板，也可以轻松地学会如何使用翼龙面板进行管理。

完全免费

与其他付费的游戏面板相比，翼龙面板最大的优势就在于完全免费、开源，您不再需要为了面板而多花一分钱。

可扩展性

无论您是用于商业化服务器出租，还是个人开服，翼龙面板都能为您提供全方面的技术支持，详情请访问社区。
